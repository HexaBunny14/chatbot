# 🎬 Entertainment Suggestion Chatbot

## 📌 Description

This project is a chatbot that provides entertainment suggestions like movies, web series, music, and games based on user input. It helps users quickly decide what to watch or listen to.

## 🚀 Features

* Suggests movies, series, and songs
* User-friendly chat interface
* Fast and interactive responses
* Category-based recommendations

## 🛠 Technologies Used

* HTML, CSS, JavaScript
* Node.js (if backend used)
* API integration (if any)

## ▶️ How to Run

1. Clone the repository
2. Run `npm install`
3. Start server using `node server.js` or `npm start`
4. Open in browser
